# Calculadora de Comida

Calculadora sencilla en HTML para sumar cuentas de un puesto de comida.

## Uso

Abre `calculadora_comida.html` en el navegador y captura las cantidades de cada producto. El total se actualiza con los valores ingresados y muestra una cantidad adicional discreta para pagos con tarjeta.

## Funciones

- Tacos, pata y refresco.
- Consomés por chico, medio y litro.
- Tortillas por kilo, medio kilo o monto directo en pesos.
- Kilo de carne por fracciones.
- Cargo opcional para llevar.
- Total principal y monto con 3% para tarjeta redondeado hacia arriba.
